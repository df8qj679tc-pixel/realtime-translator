# 即时英文翻译助手

Windows 桌面小工具：打开后监听鼠标拖选英文，抓取选中文本，翻译成中文，并给出基础语法解释。

## 功能

- 鼠标拖选英文后自动翻译为中文。
- 支持 `Ctrl+Shift+T` 手动抓取当前选中文本。
- 提供句子类型、时态、语态、情态动词、从句、短语结构等规则解释。
- 默认窗口置顶，便于边读边看。
- 默认恢复原来的剪贴板文本，减少打扰。

## 运行

需要 Windows 和 Python 3。

```bat
python realtime_translator.py
```

也可以双击：

- `run_translator.bat`
- `start_translator_silent.vbs`

## 说明

翻译使用在线接口，需要联网。语法解释是本地规则分析，适合阅读辅助，复杂句子仍建议人工判断。

仓库只保存源码和启动脚本，不保存本地打包生成的 `.exe`、`.zip` 或缓存文件。
